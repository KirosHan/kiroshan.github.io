video标签底边无法完整填充

发表于 2018-03-23 | 分类于 Html |

问题描述

使用video标签时，视频无法完整填充父级容器，底部留有空边。这是我使用视频当做背景时遇到的问题，无论设置自适应和百分比均无法解决。

原因解析

该bug疑与父元素样式background属性有关，详细原因未知

解决方案

1.父元素加上 font-size:0;
2.父元素去掉background/background-color属性
