微信公众号后台开发踩过的坑

发表于 2017-08-03 | 分类于 Laravel |

0x00 开发环境

  • Ubuntu server 16.04
  • Apache 2.4
  • PHP 7.0
  • Mysql 14.14
  • Laravel 5.4

0x01 问题一 报错”Undefined index: HTTP_RAW_POST_DATA”

原因：PHP已废除 HTTP_RAW_POST_DATA
解决方案：替换为$postStr = isset($GLOBALS['HTTP_RAW_POST_DATA']) ? $GLOBALS['HTTP_RAW_POST_DATA'] : file_get_contents("php://input");
